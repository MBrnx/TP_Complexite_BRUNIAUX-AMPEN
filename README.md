# TP_Complexite_BRUNIAUX-AMPEN

## TP1 - 
### Hanoï 
Pour l'exécution du programme de Hanoï, la seule chose à faire est de changer la valeur de **_n_** (`int n = 15;`) dans le main pour correspondre au nombre de disque souhaité <br>
Pour ne pas avoir toute les phases de modification de **Hanoï**, il suffit de commenter : <br>
`// printf("Déplacer le disque %d de la tour %c à la tour %c\n", n, A, C);` <br>
`// printf("Déplacer le disque 1 de la tour %c à la tour %c\n", A, C);`
